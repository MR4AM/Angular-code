# 基于Angular搭建的前端组件化市场
## angular安装搭建
### npm install -g @angular/cli
## 创建项目目录
### ng new angualr-app
## 运行项目创建运行服务
### ng serve --open

- 自动生成组件，到指定组件目录下下执行以下命令
- ng generate component component-name

- Angular 2 指令生命周期钩子的作用及调用顺序
- 基于指令与组件的区别来分类:
 1、指令与组件共有的钩子：
ngOnChanges
ngOnInit
ngDoCheck
ngOnDestroy
2、组件特有的钩子
ngAfterContentInit
ngAfterContentChecked
ngAfterViewInit
ngAfterViewChecked
